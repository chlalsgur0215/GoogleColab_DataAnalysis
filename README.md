# GoogleColab_DataAnalysis

구글 코랩을 활용한 데이터 분석
주제 : 웹 크롤링을 활용한 연도별 물가상승률과 최저임금 상승률의 관계 분석 및 예측

목적 : 배운 데이터 분석을 통해 웹 크롤링과 OpenAPI를 활용해 실제 최저임금과 물가상승률 데이터를 수집하고, 이를 통해 두 데이터 간의 상관관계를 분석.
 ㄴ 상관관계 분석, 선형회귀 분석, 주성분 분석, 시계열 분석(ARIMA)

 순서
 데이터 생성 ( 웹 크롤링, OpenAPI)
 데이터 가공, 정제 (이상치 처리, 타입 변경, 결측치 처리)
 상관관계 분석 (피어슨, 스피어만, 켄달)
 선형회귀 분석 (독립변수 : 물가상승률 / 종속변수 : 최저임금 상승률) + 시각화 + 최저임금 상승률 예측 및 결정계수 확인
 다중 선형회귀 분석(독립변수에 연도를 추가) + 3D plot 시각화 + 최저임금 상승률 예측 및 결정계수 확인
 주성분 분석
 시계열 분석(ARIMA모델 활용) + 예측 시각화
 선형회귀, 시계열 분석에 대한 차이점 비교분석
